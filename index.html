Smart Waste Classifier

Project Goal

People often mix up waste, leading to contamination in recycling bins and more trash ending up in landfills. This project uses machine learning to automatically detect four types of waste: Paper, Plastic, Metal, and Organic Waste.

How to Use the Classifier

  1. Click the Start button.
  2. Allow the website to access your webcam.
  3. Hold any object (paper, plastic, metal, or organic waste) in front of the camera.
  4. The model will detect the object and show the predicted category below.
